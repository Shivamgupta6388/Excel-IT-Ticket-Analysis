# IT Support Performance Dashboard | Excel Project

An interactive Excel dashboard analyzing **97,498 IT support tickets** (2016–2020).

![IT Support Performance Dashboard](images/it-support-dashboard.png)

*Fully interactive dashboard with multiple slicers*

## 🚀 Key Highlights

• Ticket volume grew by **123%** (13,051 → 29,088)  
• Average resolution time stable at **4.55 days**  
• Login Access resolved in just **0.31 days** ⚡  
• Hardware took the longest — **7.63 days** (major bottleneck)  
• System issues averaged **6.62 days**  
• 50,770 tickets received perfect **5/5** satisfaction

## 🛠️ Features

- Interactive Slicers (Year, Category, Severity, Priority, Issue Type)  
- Dynamic Pivot Tables & Charts  
- Agent performance analysis  
- Satisfaction & Resolution time trends

## 💡 Recommendations

**Priority Order:**  
1. Upgrade Ticket Management System (Highest ROI)  
2. Targeted Training for underperforming agents  
3. Hiring only if needed after optimization
